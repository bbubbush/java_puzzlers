# 18번째 퍼즐 - String cheeze

```{.java}
public class StringCheeze {
	public static void main(String args[]) {
		// Note: \u000A is Unicode representation of linefeed (LF)
		char c = 0x000A;
		System.out.println(c);
	}
}
```
